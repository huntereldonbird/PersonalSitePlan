The Website Idea

Originally, we wanted a system to compare wins and losses, so we started keeping track of our games to compete with each other over a larger period of time and in different games. So we have a Google doc with a large data set of games and statistics, but the problem is that it is impossible to quickly figure out what you are looking for. The information is well organized, it is just sometimes WAY too much information to look through. So my website will work like the MLB or any other sports league website where it will quickly show recent games and player stats. It will let us all see the current ranking for each major game that we play, and the scheduled time and dates. I plan on doing this by learning some PHP and maybe trying to use that as the database, failing that, using JavaScript to have it function.

Modding Info Section

I wanted to include another page about our game's modding situation. We have had a lot of problems keeping track of which mod and “playset” we are going to be using, and we end up wasting way too much time when a few players can't figure it out quickly. I thought that if we had a quick way to access the link in the same place that you look at the other stats, that would make it a little more obvious where to download it and how to get in the game quicker.

The second issue I wanted to solve was the disconnect between the mods and the players. We have a lot of information already added to a spreadsheet online that anyone can look at. This spreadsheet tells you All of the additional features and in-game statistics that are added by our mods. So by going through them, you can form a more complex strategy around newly changed In-game statistics. I wanted to add this because I have personal experience as the person who makes the mods for our Stellaris campaigns. Since I make the mod and add all the info into the spreadsheet, I usually have an advantage when I’m playing because I’m more comfortable with the new changes. Because I know not everyone looks through those because they can sometimes be hard to find, I wanted to put them somewhere where they can quickly referenced.
